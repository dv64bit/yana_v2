<!-- GETTING STARTED -->
## Getting Started

Install the node dependency
* npm
  ```sh
  npm install
  ```

### Running

After installing the dependencies
* Initialise the application:
   ```sh
   nodemon
   ```
* Now go to browser and search
   ```sh
   localhost:3000
   ```
